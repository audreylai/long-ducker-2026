# Lion Auction site for Long Ducker

## Tech stack
- Flask
- Tailwind CSS
  - Colour scheme should follow branding of Harrow International School Hong Kong
- MongoDB (Can expand if needed)
  - Lion collection
  - Bids collection
- SocketsIO

## Pages
- Admin
  - Auction Management
    - Create/edit/view Bids
  - Manage Lions
  - Export data to CSV
- User
  - View Lions
  - Place Bids (fill in form for bid amount and contact details)
  - Live update
